## 📌 Project: Uzhavan AI
**Uzhavan AI** is an intelligent agricultural prototype built to assist rural farmers with real-time crop disease diagnosis using the **Gemini API**. This project was recognized as a **Hackathon Winning Project** for its innovative use of AI in social sectors.

---

## 🏆 Key Achievements
*   **Hackathon Winner**: Recognized for technical feasibility and agricultural impact.
*   **ISTE Best Student Award (2026)**: Awarded for technical innovation and leadership.
*   **IEEE YESIST12 Participant**: Showcased at an international innovation forum.

---

## 🛠️ The Vibe Coding Tech Stack
*   **Brain**: **Google Gemini Vision AI** for automated crop disease diagnosis.
*   **Logic**: Python-driven backend with a local **uzhavan_database.db** for data persistence.
*   **Rapid UI**: Developed using **Streamlit** for instant deployment.
*   **Accessibility**: **gTTS (Google Text-to-Speech)** integration for voice-over advice in regional languages.

---

## 📂 Project Structure
*   📄 `app.py`: Main Streamlit application logic.
*   📄 `uzhavan_database.db`: Local SQLite database for agricultural metadata.
*   📄 `requirements.txt`: Environment dependencies for rapid deployment.
